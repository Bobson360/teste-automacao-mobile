## Objetivo

automatizar uma operação na calculadora android
 
## Pré-Condições

1. Utilize Java
2. Organize o código utilizando Page Object
3. Utilize BDD (Cucumber)
4. Appium
5. TestNg ou Junit


### Cenário a ser automatizado

* Abra a calculadora do android
* faça uma operação qualquer
* valide o resultado
